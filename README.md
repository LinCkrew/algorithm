## 📌 규칙

1. 각자의 풀이는 `[주차]/problem[문제번호]/[이니셜].js` 형식으로 업로드
2. 파일명은 이니셜로 통일 (예: njh.js, ksw.js)
3. 각 문제 폴더에는 README.md 파일에 문제 내용 기재

## 🛠 개발 환경

## 💻 시작하기

